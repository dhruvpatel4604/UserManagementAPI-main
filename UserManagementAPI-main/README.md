# User Management API

A simple ASP.NET Core Web API for managing users.

## Features

- CRUD operations for users
- Input validation
- Logging middleware
- Error handling middleware
- Swagger UI for testing

## Microsoft Copilot Contribution

Microsoft Copilot was used during development to assist with:

- Generating project boilerplate
- Creating CRUD endpoints
- Debugging controller logic
- Improving validation checks
- Implementing middleware
- Enhancing code readability

## Endpoints

GET /api/users  
GET /api/users/{id}  
POST /api/users  
PUT /api/users/{id}  
DELETE /api/users/{id}
